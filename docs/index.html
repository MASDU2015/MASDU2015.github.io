马协简介

山东大学马拉松爱好者协会#

山东大学马拉松爱好者协会是山东大学的群众性体育类社团。成立于2015年5月4日，是学校广大跑步爱好者的活动平台，协会创办目的是希望以马拉松长跑锻炼为纽带，通过日常的体训和丰富多彩的跑步活动，使更多的同学养成科学的锻炼习惯，并从跑步中获得快乐，收获成长。
